# CIFAR10 对比实验结果

## 实验配置

### 核心训练参数

| 参数 | 值 | 意义 |
|-----|---|------|
| `--dataset` | CIFAR10 | 数据集选择（10类彩色图像，32x32） |
| `--num_clients` | 10 | 客户端数量（模拟10个独立数据源） |
| `--global_epoch` | 40 | 全局训练轮次（服务器聚合次数） |
| `--local_epoch` | 2 | 每个客户端本地训练轮次 |
| `--lr` | 0.001 | 学习率（梯度更新步长） |
| `--batch_size` | 16 | 批次大小（每次梯度计算样本数） |

---

### 差分隐私参数

| 参数 | 值 | 意义 |
|-----|---|------|
| `--dp_method` | dfl/gaussian | DP方法选择 |
| `--target_epsilon` | 1.0 | 隐私预算 ε（越大保护越弱） |
| `--clipping_bound` | 0.5 | 梯度裁剪边界（限制梯度范数） |
| `--sparsity_ratio` | 0.4 | 梯度稀疏化比例（40%梯度置零） |

---

### DFL 混沌噪声参数

| 参数 | 值 | 意义 | 范围 |
|-----|---|------|------|
| `--dfl_mu` | 3.95 | **混沌控制参数**，控制混沌程度 | [3.57, 4.0] |
| | | • 3.95 = 接近全混沌 | |
| | | • 越高 → 混沌性越强，噪声越不可预测 | |
| `--dfl_alpha` | 0.85 | **分数阶参数**，控制记忆效应 | (0, 1] |
| | | • 0.85 = 中等记忆性 | |
| | | • 越低 → 更多历史依赖，混沌性减弱 | |
| `--chaotic_factor` | 0.4 | **混沌噪声强度因子**，DFL噪声占比 | [0, 1] |
| | | • 0.4 = 40%DFL + 60%混合噪声 | |
| | | • 越高 → 混沌噪声越强，安全性越高 | |
| `--chaotic_decay` | 0.9 | **混沌衰减率**，每轮次衰减 | (0, 1] |
| | | • 0.9 = 每轮混沌强度降低10% | |
| | | • 用于平衡训练后期安全性 vs 准确率 | |

---

### 噪声缩放参数

| 参数 | 值 | 意义 |
|-----|---|------|
| `--sigma_factor_dfl` | 0.002 | DFL噪声乘数，控制噪声强度 |
| `--sigma_factor_gaussian` | 0.002 | 高斯噪声乘数，控制噪声强度 |

---

### 参数关系图

```
梯度裁剪 (clipping_bound)
        ↓
    梯度值
        ↓
    噪声注入 (dp_method)
        ├─ Gaussian: 标准正态噪声
        └─ DFL: 混沌噪声 (dfl_mu × dfl_alpha × chaotic_factor)
        ↓
    梯度稀疏化 (sparsity_ratio: 40% → 置零)
        ↓
    聚合 → 全局模型更新
```

---

### 推荐参数组合逻辑

**mu=3.95, alpha=0.85, cf=0.4** 的理由：
1. **mu=3.95**：足够混沌（不可预测），但不极端（3.99），保持一定稳定性
2. **alpha=0.85**：适度记忆性，平衡混沌扩散与收敛速度
3. **cf=0.4**：混沌噪声占40%，既提供安全性又不过度干扰梯度

这组参数在 MNIST 和 CIFAR10 上都验证了综合优势。

### 完整命令

#### DFL 方法
```bash
python ours.py \
    --dataset CIFAR10 \
    --dp_method dfl \
    --global_epoch 40 \
    --local_epoch 2 \
    --num_clients 10 \
    --lr 0.001 \
    --batch_size 16 \
    --target_epsilon 1.0 \
    --clipping_bound 0.5 \
    --sparsity_ratio 0.4 \
    --dfl_mu 3.95 \
    --dfl_alpha 0.85 \
    --chaotic_factor 0.4 \
    --chaotic_decay 0.9 \
    --sigma_factor_dfl 0.002 \
    --store
```

#### Gaussian 方法
```bash
python ours.py \
    --dataset CIFAR10 \
    --dp_method gaussian \
    --global_epoch 40 \
    --local_epoch 2 \
    --num_clients 10 \
    --lr 0.001 \
    --batch_size 16 \
    --target_epsilon 1.0 \
    --clipping_bound 0.5 \
    --sparsity_ratio 0.4 \
    --sigma_factor_gaussian 0.002 \
    --store
```
